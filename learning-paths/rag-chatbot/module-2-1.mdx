## Step 1: Clone the Repository

<Steps>
  <Step title="1. Navigate to your development directory">
    Open your terminal and navigate to where you want to create the project
  </Step>
  <Step title="2. Clone the repository">
    Use git to clone the starter repository
  </Step>
  <Step title="3. Navigate into the project directory">
    Change into the newly created project folder
  </Step>
</Steps>

<Snippet>
```bash
# Navigate to your development directory
cd ~/projects

# Clone the repository

git clone https://github.com/vercel/ai-sdk-rag-starter.git

# Navigate into the project

cd ai-sdk-rag-starter

````
</Snippet>

<Callout type="warning">
  **Note**: Make sure you have Git installed on your system. If not, install it from [git-scm.com](https://git-scm.com/)
</Callout>

---

## Step 2: Install Dependencies

<Steps>
  <Step title="1. Install Node.js dependencies">
    Install all required packages using npm or yarn
  </Step>
  <Step title="2. Verify installation">
    Check that all dependencies are installed correctly
  </Step>
</Steps>

<Snippet>
```bash
# Install dependencies
npm install

# Or if you prefer yarn
yarn install

# Verify installation
npm run build
````

</Snippet>

<Callout type="info">
  **Package Manager**: The project uses npm by default, but you can use yarn or
  pnpm if preferred
</Callout>

## Step 3: Environment Configuration

<Steps>
  <Step title="1. Copy environment template">
    Create your local environment file from the template
  </Step>
  <Step title="2. Configure API keys">
    Add your OpenAI API key and database connection string
  </Step>
  <Step title="3. Verify configuration">
    Ensure all required environment variables are set
  </Step>
</Steps>

<Snippet>
```bash
# Copy the environment template
cp .env.example .env.local

# Edit the environment file

nano .env.local

````
</Snippet>

### Required Environment Variables

<CodeGroup>
  <CodeGroupItem title="OpenAI Configuration">
```bash
# OpenAI API Key (required for embeddings and chat)
OPENAI_API_KEY=sk-your-openai-api-key-here

# OpenAI Model (optional, defaults to gpt-4o)
OPENAI_MODEL=gpt-4o
````

  </CodeGroupItem>
  <CodeGroupItem title="Database Configuration">
```bash
# PostgreSQL connection string
DATABASE_URL=postgresql://username:password@localhost:5432/rag_chatbot

# Database URL for Vercel (production)

POSTGRES_URL=postgresql://username:password@localhost:5432/rag_chatbot

````
  </CodeGroupItem>
</CodeGroup>

<Callout type="warning">
  **Security**: Never commit your `.env.local` file to version control. It's already in `.gitignore`
</Callout>

## Step 5: Verify Setup

<Steps>
  <Step title="1. Check Node.js version">
    Ensure you're using Node.js 18 or higher
  </Step>
  <Step title="2. Verify dependencies">
    Confirm all packages are installed correctly
  </Step>
  <Step title="3. Test build process">
    Ensure the project can be built without errors
  </Step>
</Steps>

<Snippet>
```bash
# Check Node.js version
node --version  # Should be 18.0.0 or higher

# Check npm version
npm --version

# Verify dependencies
npm list --depth=0

# Test build
npm run build
````

</Snippet>

<Callout type="info">
  **Success Indicators**: - No dependency warnings - Build completes without
  errors - All TypeScript types are valid
</Callout>

### Troubleshooting Common Issues

<Callout type="warning">
  **Issue**: `npm install` fails with permission errors **Solution**: Use `npm
  install --legacy-peer-deps` or check your Node.js version
</Callout>

<Callout type="warning">
  **Issue**: TypeScript errors during build **Solution**: Ensure you have the
  correct TypeScript version and all type definitions
</Callout>

<Callout type="warning">
  **Issue**: Environment variables not recognized **Solution**: Restart your
  development server after adding environment variables
</Callout>
