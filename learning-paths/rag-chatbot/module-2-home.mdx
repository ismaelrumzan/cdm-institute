## Learning Objectives

By the end of this module, you will be able to:

- Clone and set up the Vercel AI SDK RAG Starter repository
- Configure PostgreSQL database with pgvector extension
- Set up development environment with proper tooling
- Understand the project structure and architecture

## Module Overview

This module guides you through setting up your development environment and getting the Vercel AI SDK RAG Starter project running locally. You'll configure the database, install dependencies, and understand the project structure.

<Callout>**Time Estimate**: 95 minutes total</Callout>

## Prerequisites

Before starting this module, ensure you have:

- Git installed on your system
- Node.js 18+ installed
- A code editor (VS Code recommended)
- Basic familiarity with command line tools
- No prior experience with Next.js or Vercel required
